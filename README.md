# Personal Portfolio Website

A responsive and accessible personal portfolio website built using
HTML, CSS, and JavaScript.

## Features
- Semantic HTML
- WCAG 2.1 AA accessibility friendly
- Keyboard navigable
- Clean and responsive UI
- W3C HTML & CSS validated

## Technologies Used
- HTML5
- CSS3

## Accessibility
This project follows accessibility best practices including
proper contrast ratios, ARIA labels, and focus-visible styles.
